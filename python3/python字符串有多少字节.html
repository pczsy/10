    • python字符串有多少字节
    原文出处:本文由博客园博主python之蝉提供。
    原文连接:https://www.cnblogs.com/pythonzhichan/p/11406839.html

    是否有一些函数可以告诉我字符串在内存中占用多少字节？

    我需要设置套接字缓冲区的大小，以便一次传输整个字符串。

     

    解决方案

    import sys
sys.getsizeof(s)

# getsizeof(object, default) -> int
# Return the size of object in bytes.

    但实际上你需要知道它代表的长度，所以类似的东西len(s)应该足够了。
