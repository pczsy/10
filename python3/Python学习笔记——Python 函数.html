    10. 闭包

    #如果在一个内部函数里，对在外部作用域（但不是在全局作用域）的变量
#进行引用，那么内部函数就被认为是闭包(closure)。
def FunX(x):
    def FunY(y):
        return x*y
    return FunY
i = FunX(8)
print(type(i))

print(i(5))
print(FunX(8)(5))
