        最近想系统的学习一下爬虫，就在网上找了崔大大《Python3 网络爬虫开发实战》，然后。。。单配置开发环境就用了将近一天的时间，踩到了各种各样的，奇形怪状的坑。把能想到的再总结一下。

    一、 1.4.1 MYSQL的安装

        1、登陆MySQL官网：https://www.mysql.com/cn/

        2、点击下载并下滑到页面最底部

        3、点击 MySQL Community (GPL) Downloads

    4、点击 MySQL Community Server

    5、点击 GO to Download Page >

        6、选择第二个并下载

        64位系统的也是选择这个下载，我一开始以为没有64位的包，然后下载了需要自己配置的非安装版，发现不容易配置成功（脸太黑），在网上搜索了一下，然后说的是64位的也下载这个就可以了。如果之前安装过MySQL需要先将之前的彻底删干净（包括注册表）
