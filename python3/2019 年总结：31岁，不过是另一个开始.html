    • 2019 年总结：31岁，不过是另一个开始
    原文出处:本文由博客园博主王磊的博客提供。
    原文连接:https://www.cnblogs.com/vipstone/p/12081332.html

    多年前曾看过一个关于黄磊的采访，其中大部分的内容已经忘记的差不多了，其中有一句话一直记忆深刻，他说：“人生有很多阶段，每一个阶段都有它的快乐和苦涩，但这些你无法跨过去也不能退回来，你要做的就是享受每一个阶段所特有的快乐”，那一刻犹如醍醐灌顶，让我记忆至今。

    29 岁之后，开始觉得人生没有方向，不知道多年后的自己该去往何处归往何方，迷茫与焦虑的情绪与日俱增。于是就出现了 2017 年的跨年，我独自在嘉兴乌镇旁边的一个小镇，2018 年的跨年我又在上海，今年的跨年又回到了西安。值得庆幸的事，今年终于找到了新的人生方向。

    1 人生的两次重大选择

    31 年之前，我人生中最重要的两次选择，一个是学程序，另一个是坚持写作。

    选择程序开发，能让我填饱肚子的同时又有一份不错的收入，而写作则让我确认了人生的方向和最终的归途。程序的开发之路平淡无奇，像大多数人一样坚持了 10 几年，而多年没有任何回报的写作，却在迷茫时为我打开了另一扇门。

    2 无意之中的一篇爆款

    我写博客是从 2010 年开始的，到现在已经有 9 年多了，刚开始是为了方便记录一下技术点，到后面就越写越多，但几乎没什么收入，偶尔一次五毛或一块的打赏，会让我高兴好一阵子。

    然而事情的转机出现在今年的三月份，今年三月初我在平发布了一篇关于 Java 面试的付费文章「Java 最常见 200+ 面试题全解析」，竟然意外的成为了平台的爆款，短短的几个月内就变成了平台单篇付费的头部作品，之后销量一路飙升，截止目前为止，已经卖出了 11366 份，售价是 9.9 元，这次的成功给了我很多的信心，之后总结了这次成功的原因，趁热打铁又迅速推出了三个专栏，一月份还有一个重磅的课程推出。一路走来也很艰辛，所有人看到的都是你成功的那一刻，但往往过程才是最煎熬的，只因自己的兴趣使然，才一步步的走到了今天，不能说取得了多大的成就，起码鼓舞了我的信心，能让我在这条路上坚持不懈的走下去。

    3 今年的收获

    1）写作方面

    今年几乎所有的业务时间都投入到写作这件事上了，码了大概 40 万的字，加上马上要发布的专栏，总共是四个专栏了，也算小有所获，但相比于收入来说，能用自己的力量让更多的人受益，显然更能让我感觉到快乐，这也是我一个恒久不变的信念(服务更多的技术人)。

    已经发布的专栏分别是：

    • 《Java 面试全解析》Java 核心知识点与面试题：共有 15 万字和 500 多道面试题；
    • 《程序员的 MySQL 面试宝典》以面试出发，深入和全面的介绍了 MySQL 的一些底层结构和原理。
    • 《设计模式 34 讲》案例以 Java 为主，是我目前看过最好的关于设计模式的课程。

    为了避免广告之嫌，我就不放链接地址了。

    2）微信公众号运营

    公众号「Java中文社群」从去年年底开始做，中间因为有一段时间专心去做专栏了公众号几乎没怎么运营，所以到目前为止的关注量只有一万多一点，阅读量也不是很高通常不到一千，后面会持续运营起来，基本可以做到每周一到周五都会更新文章，最近一段时间都在连载 Java 面试的文章，有需求的小伙伴可以关注一波。

    3）读书

    今年读书没有像往年一样，给自己订一个数量，都是用到什么书，看什么书，买纸质书在当当的消费也快超过了 1000 大洋了。

    在读书这块，我的感悟是，用世界上公认学习最快的学习法费曼学习法，用输出倒闭输入的方式，让自己更快的提升。

    4 今年的感悟

    以下的几点来自于今年的感悟，希望分享给大家，欢迎评论区留言补充和讨论。

    1）没有完美的公司

    之前看过一个统计，关于全球平均在职时间的排行，其中 Google 的平均在职时间是 3.2 年，其他的公司都在 3 年以内，那么问题来了，如果有完美的公司，为什么离职了会这高？平均在职时间怎么短？所有说没有完美的公司，每个公司都要自己缺点，不要总是抱怨，因为这样做基本没什么用。如果实在不想呆了，就骑驴找马做点实际的事，对将来有帮助的事才是最重要的。

    2）没有理想的条件和环境

    我们做任何事，都想要理想的条件和理想的环境，基本是不存在的，就好比跑步之前总想要有好的跑鞋，要好的运动服才能开始，当买到所有的装装备之后，你会发现，你缺少的真不是一两件装备而已，同样对理想条件的执念也是一样，所以要学会与生活共舞和“不确定”一起上路。

    3）坚持做一件长期有益的事

    人们通常容易做一件有短期收益的事，比如吃一顿美食，看一档喜剧电影，这些能让你短时间内的获得快乐和满足，但对你日后的生活未必有帮助，25 岁之后要持续做一件对自己长期有意的事，例如锻炼身体、读书、写作等，这些会在多年之后给你带来意想不到的惊喜。

    4）渡人先渡己

    子曰：行有余力，则以学文，渡人先渡己，先解决温饱问题再追求梦想，但一定要记得后者，所谓念念不忘必有回响，不单单只是一句鸡汤。

    5）最核心的能力：自我进化能力

    这个时代最核心的能力就是自我进化的能力，一个人最可怕的能力就是持续的学习力，愿你在日懂得，并付诸实践。

    6）收获个人商业模式

    是的，个人也可以有商业模式，把自己当成一个公司来运营，一般个人的商业模式有四种：

    • 一份时间出售一次，例如上班，去上班了就有工资拿，不上班就没有工资；
    • 一份时间出售多次，例如写作，影视，用一次时间创造之后，可以持续的售卖；
    • 打包别人的时间出售，例如雇人做产品，在靠产品转更多的钱；
    • 用钱生钱，例如理财。

    以上为个人今年的感悟，也欢迎更多的小伙伴添加我个人的微信：GG_Stone （备注2020），相互交流共同成长。

      做好自己，把其余的交给时间，要相信时间所带来的复利，坚信量变才会引起质变的基本规律。种一棵树，最好的时间是十年前，其次是现在。加油~
