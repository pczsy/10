    windows下使用pip提示ImportError: cannot import name 'main'

    原因：将pip更新为10.0.0后库里面的函数有所变动造成这个问题

    解决方法：先卸载现在的pip:

    python -m pip uninstall pip

    然后重装：python -m pip install pip==19.0.3

     windows下虚拟环境的安装：

    在命令窗口下使用pip install virtualenv 安装

    安装成功后继续使用pip install virtualenvwrapper-win

    配置好环境变量后，在命令窗口输入lsvirtualenv提示'lsvirtualenv' 不是内部或外部命令，也不是可运行的程序
    或批处理文件。

    原因：没有安装virtualenvwrapper-win，导致命令无法使用
