    针对windows操作系统

    1、下载安装包，python官网下载（https://www.python.org/）

    python3.7.4：下载https://www.python.org/downloads/release/python-374/，选择Windows x86-64 executable installer

    python2.7.16：下载https://www.python.org/downloads/release/python-2716/，选择Windows x86-64 MSI installer

     

    2、安装

    直接双击运行，python3.7.4选择customer install

     

    3、设置环境变量

    我的电脑->属性->高级系统设置->环境变量->系统变量->path

    在path中添加python安装的python.exe所在全路径

     

    4、验证python命令是否成功

    打开shell，输入python查看python版本
