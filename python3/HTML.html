    一，HTML

    1. HTML 超文本标记语言

      1. 超文本：图片、视频、音频

      2. 标记：标签、标记

    1. 标记的分类：

      1. 双边标记，又称'双封闭标签'

      2. 单边标记，又称'单封闭标签'

    2. 标签的分类

      1. 块级标签

        h1 - h6

      1. 行内（内联）标签

        span
        br                  换行
        b 或 strong          加粗
        i 或 em              斜体
        s 或 del             删除线
        u                   下划线
        img                 图片
            src 来源
                1.网络地址
                2.本地路径
            alt 图片的提示
            width 宽度
            height 高度
        a 或 anchor          锚 / 超链接
            href 网址
                # 回到顶部
                页面锚点：
                        1.标签添加id
                        2.a标签加name
            target
                _self(默认)   当前页面刷新
                _blank      新建一个窗口
        ​
        标签
            head  相当于一个人的思维，存在，但是看不见
                meta 元信息
                    charset 编码
                    <meta http-equiv="refresh" content="2;URL=http://www.luffycity.com">
                    description 描述
                    Keywords 关键字
                title 标题
                    <style></style>  <!-- css -->
                    <link rel="stylesheet" href=""> <!-- 连接 -->
                    <script></script>    <!-- js -->
            body  相当于一个人的身体，可以看见，可以摸到
