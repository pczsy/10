    • python面试题之什么是Tkinter ？

    TKinter是一款很知名的Python库，用它我们可以制作图形用户界面。其支持不同的GUI工具和窗口构件，比如按钮、标签、文本框等等。这些工具和构件均有不同的属性，比如维度、颜色、字体等。

     

    >>> import tkinter
>>> top=tkinter.Tk()
