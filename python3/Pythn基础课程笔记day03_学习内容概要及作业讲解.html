    作业回顾

    1、猜数字，设定一个理想数字比如：66，让用户输入数字，如果比66大，则显示猜测的结果大了；如果比66小，则显示猜测的结果小了；只有等于66，显示猜测结果正确，然后退出循环。

    while True:
    num = input('请输入数字：')
    num = int(num)
    if num > 66:
        print('猜测的结果大了')
    elif num < 66 :
        print('猜测的结果小了')
    else:
        print('猜测的结果正确')
        break

    2、在上一题的基础，设置：给用户三次猜测的机会，如果三次之内猜测正确，则显示猜测结果正确，退出循环，如果三次之内没有猜测正确，则自动退出循环，并显示“你是个大笨蛋”。

    # 方式一
count = 1
while count <= 3:
    num = input('请输入数字：')
    num = int(num)
    if num > 66:
        print('猜测的结果大了')
    elif num < 66:
        print('猜测的结果小了')
    else:
        print('猜测的结果正确')
        break
    count += 1
else:
        print('你是个大笨蛋')

    3、 用户三次登录，登录成功退出，失败三次结束

    count = 1
while count <= 3:
    user = input("请输入用户名：")
    pwd = input("请输入密码：")
    if user == '张磊' and pwd == '123456':
        print("恭喜!登录成功")
        break
    else:
        print("抱歉!登录失败")
    if count == 3:
        break
    count += 1

    4、登录三次机会提示

    count = 1
while count <= 3:
