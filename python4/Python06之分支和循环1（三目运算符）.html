    Python 为了使得程序更加简洁而新引入过来的一个三目操作符，顾名思义就是有三个参数。

    　　格式： x if 条件表达式 else y

    　　　　先判断条件表达式真假，真则取 x 的值，否则取 y 的值。

    　　　　例如： 3 if 4>3 else 4 值为 3，而 3 if 4<3 else4 的值为 4 

    x = 4
    y = 5
    z = "x小于y" if x<y else "x大于y"
z
'x小于y'
z = "x小于y" if x>y else "x大于y"
z
'x大于y'
