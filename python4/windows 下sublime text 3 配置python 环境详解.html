    • windows 下sublime text 3 配置python 环境详解

    这我们的环境已经安装了python 3.7.1解释器和sublime text 3 编辑器

    一、package control 安装

    1.   首先我们打开sublime text 3 ——>Tools——>Install Package Control，等几分钟就弹出提示窗体，表示安装成功
    2.   然后我们使用的时候发现，报错“no packages”，大概意思是有一个获取package网址不可用，也就是channel_v3.json没用

                Preferences——> Package Settings——>Package Control——>Settings-user，并添加如图所示代码

                 ​

                 

    "channels": [
        "https://raw.githubusercontent.com/Bouquet666/Universal-tool/master/channel_v3/channel_v3.json"
    ],

                保存，重启sublime text 3。Package Control 可用

    二、汉化sublime text （ChineseLocalizations）

    1. 快捷键“ctrl+shift+p”——>输入“install”,然后点击图中所示位置                                            ​
    2. 然后输入“ChineseLocalizations”，回车安装，等一会儿就直接安装了语言包，最后切换语言在“help”——>“Language”，切换你的语言

     三、修改默认的主题配色设置（PackageResourceViewer）

    1. 快捷键“ctrl+shift+p”——>输入“install”,然后点击图中所示位置                                            ​
    2.  然后输入“PackageResourceViewer”，回车安装，等一会儿就好了。
    3.  打开工具栏——>工具——>命令面板，输入PackageResourceViewer:Open Resource，回车安装​
    4. 安装PackageResourceViewer:Open Resource成功后，在输入框里找到sublime的默认主题，点击 Color Scheme -Default。然后你就可以选择你想要的额。最后如果回到默认主题，就在setting-user中删除配置的主题。 

    四、 实现python交互功能（SublimeREPL）

    1. 快捷键“ctrl+shift+p”——>输入“install”,然后点击图中所示位置                                             ​
    2.  然后输入“SublimeREPL”，回车安装。
    3. 最后配置快捷键，点击Sublime Text 3菜单栏Preferences——>Key Bindings。   ​
    4. 在新弹出的窗口中，在Default(Windows).sublime-keymap--User中输入如下的代码，保存后退出 
          {
        "keys": ["f5"],
        "caption": "SublimeREPL: Python - RUN current file",
        "command": "run_existing_window_command",
        "args": {
            "id": "repl_python_run",
            "file": "config/Python/Main.sublime-menu"
        }
    },
    {
        "keys": ["f8"],
        "caption": "SublimeREPL: Python - PDB current file",
        "command": "run_existing_window_command",
        "args": {
            "id": "repl_python_pdb",
            "file": "config/Python/Main.sublime-menu"
        }
    },
       按F5运行，F8调试，记得保存先

          以上的配置都是windows 联网的状态下，无网的话，就要下好包，也就是.sublime-package文件。然后在放在Installed Packages文件夹下。如果你的电脑安装多个python 版本的话，就需要知道版本。
