    %d  digit
    %s  string
    %f  float
    程序运用：
    name = input("please input your name:")
    age = int(input("please input your age:"))
    job = input("please input your job:")
    hometown = input("please input your hometown:")
    info = '''
    ----- info of %s -----
    Name: %s
    Age : %d
    Job : %s
    Hometown: %s
    ---------- end --------
    ''' %(name,name,age,job,hometown)

    print(info)
