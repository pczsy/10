    学习资料来源

    https://github.com/jackfrued/Python-100-Days/blob/master/Day01-15/03.%E5%88%86%E6%94%AF%E7%BB%93%E6%9E%84.md

    1.应用场景

    顺序结构

    • 如，变量计算，华氏度转摄氏度，计算圆的周长和面积，判断闰年
    • 都是一条一条语句顺序执行，这种代码结构通常称之为顺序结构

    分支结构，选择结构

    • 需要在程序进入下一阶段时做出选择时用到
    • 这种结构运用于，游戏关卡结束后的选择，游戏人物升级后进行职业选择等等

    2.if语句

    关键字

    • 具有特殊含义的单词
    • 构造分支结构可以使用if、elif和else关键字

    if语句的使用

    • python中，用缩进方式来设置代码层次结构，if条件语句作为一个代码块，保持多条语句具有相同缩进即可
    """用户身份验证

    • 构造出更多的分支，可以使用if…elif…else…结构
    """
分段函数求值

        3x - 5  (x > 1)
f(x) =  x + 2   (-1 <= x <= 1)
        5x + 3  (x < -1)

Version   0.1
Author  罗绪成
"""
x=float(input('请输入自变量x='))
if x>1:
    y=3*x-5
elif -1<=x<=1:
    y=x+2
else :
    y=5*x+3
print('y=%.2f'%y)

"""
变量只能是字母数字下划线的组合
所以变量名为f（x）时

    • 在if的内部构造出一个新的分支结构，elif和else中也可以再构造新的分支，我们称之为嵌套的分支结构
    """
分段函数求值（嵌套分支结构）

        3x - 5  (x > 1)
f(x) =  x + 2   (-1 <= x <= 1)
        5x + 3  (x < -1)

Version   0.1
Author  罗绪成
"""
x=float(input('请输入自变量x='))
if x>1:
    y=3*x-5
else :
    if-1<=x<=1 :
        y=x+2
    else :
        y=5*x+3
print('y=%.2f'%y)
