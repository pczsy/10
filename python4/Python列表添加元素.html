    Python列表添加元素

    1.appent()

    　　在列表尾部添加一个元素

    　　

    >>>my_list.append("append方法")
>>>my_list
['append方法']

     

    2.extend()

    　　在列表尾部添加若干个元素

    　　

    >>> my_list.extend(["张三","李四",1,2,3])
>>> print(my_list)
['append方法', 'hello', 'hi', 'hello', 'hi', '张三', '李四', 1, 2, 3]

     

    3.insert()

    　　在列表指定下标位置出添加元素

    >>> my_list.insert(0,"测试")
>>> print(my_list)
['测试', 'append方法', 'hello', 'hi', 'hello', 'hi', '张三', '李四', 1, 2, 3]
