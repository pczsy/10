    函数：将一些语句进行封装，然后通过调用的形式，执行这些语句。

    函数的作用：

    　　1.将大量重复的语句封装在函数里，以后需要这些语句的时候，直接调用函数。

    　　2.简化编程，让编程模块化。

    函数的定义                                

    <script>
    function 函数名 () {
         
    }
</script>

     

    解释：

    　　function：定义函数的关键字。

    　　函数名：和变量的命名规则一样，只能有字母、数字、下划线、$符号组成，且不能以数字开头，不能含有保留字。

    　　参数：小括号里面放的，传递给函数的参数。

    　　大括号里面放的是函数的语句（也就是函数体）

    函数的调用                                    

    函数名（）；

     

    JavaScript中对函数的调用没有明确的规定，调用函数可以放在函数的末尾，也可放在函数的开头。
