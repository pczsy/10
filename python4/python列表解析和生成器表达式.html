    列表解析作为动态创建列表的强大工具，值得学习。

    • 列表解析技术之前的状况--函数式编程。

    　　　　lambda、filter(), map()

    　　　　enumerate, sorted, any, all, zip 等等。

    　　　　举例：list_a经过lambda函数过滤后得到列表list_b

        list_a = [1, 2, 3, 4, 5, 6]
    list_b = list(filter(lambda x: x > 3, list_a))
    print(list_b)

    　　结果: [4, 5, 6]

    • 列表解析:

    　　　　基本语法：[expr for var in iterable]

    　　　　语句的核心是for循环，迭代iterable对象的所有条目。expr作为表达式应用于迭代产生的每个成员，最后的结果值是该表达式产生的列表。

    　　　　例子:  print([x**2 for x in range(4)])

    　　　　结果为：[0, 1, 4, 9]

     　　　　扩展语法：[expr for var in iterable if cond_expr]

    　　　　表达式cond_expr作为迭代时的过滤条件。

    　　　　例子:  print([x**2 for x in range(4) if x % 2])

    　　　　结果为：[1, 9]

    • 生成器表达式:

    　　　　语法：(expr for var in iterable if cond_expr])

    　　　　生成器表达式是列表解析的扩展，与列表解析很相似（中括号换成了小括号），采用“延迟计算”方式节省内存。

    　　　　例子：print(list(x**2 for x in range(10) if x % 2))

    　　　　结果：[1, 9, 25, 49, 81]
